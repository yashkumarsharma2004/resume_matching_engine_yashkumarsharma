##Resume Matching Engine

A project to build a resume matching engine that matches resumes with job descriptions.

##Introduction

The Resume Matching Engine is a project that aims to build a system that can match resumes with job descriptions.

##Requirements
Python 3.8 or later
Git 2.24 or later

##Installation
Clone the repository: git clone https://github.com/yashkumarsharma2004/Resume_Matching_Engine_yashkumarsharma.git
Install the required dependencies: pip install -r requirements.txt

##Usage
Run the script: python resumematchingengine.py
